# 🌿 EcoTrack: Carbon Footprint Tracker

**EcoTrack** is a full-stack web application designed to help users monitor, calculate, and reduce their daily carbon emissions. Built for a 15-day "Base-to-Build" sprint, this project focuses on sustainability through data-driven insights.

---

## 🚀 The Problem & Solution
Most individuals are unaware of the environmental impact of their daily choices. **EcoTrack** provides a simple interface to input daily activities (transport, diet, energy) and receives a real-time CO2 impact score with actionable reduction tips.

---

## 🛠️ Tech Stack

### **Frontend**
* **React.js (Vite):** Functional UI components.
* **Tailwind CSS:** Modern, responsive styling.
* **Lucide Icons:** Clean iconography.

### **Backend**
* **Java (Spring Boot):** Robust REST API logic.
* **Maven/Conda:** Dependency and environment management.
* **MySql:** Secure data storage.
